# FightEngine

A comprehensive Godot 4.5 plugin for creating 2D fighting games with precision and ease.

## Overview

Fxll3n's Fight Engine (FFE) provides the essential systems and tools needed to build competitive-quality 2D fighting games in Godot. Whether you're creating a traditional arcade fighter or a platform fighter, this plugin handles the complex mechanics so you can focus on making your game unique.
